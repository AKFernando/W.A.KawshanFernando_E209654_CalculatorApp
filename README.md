# W.A.KawshanFernando_E209654_CalculatorApp

Features
- Perform **Addition**, **Subtraction**, **Multiplication**, and **Division**  
- Accepts **two numbers** as input  
- Displays accurate results immediately after pressing the “=” button  
- Handles invalid inputs gracefully (e.g., division by zero, empty inputs)  
- **Clear** button resets input and output fields  
- Clean and user-friendly **Windows Form Interface**

Tools Used
- **Programming Language:** C#  
- **Framework:** .NET Framework (Windows Forms)  
- **IDE:** Visual Studio  
- **Version Control:** Git & GitHub
